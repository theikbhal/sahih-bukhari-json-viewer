Welcome to the Hadith API

This API provides access to Sahih al-Bukhari Hadiths.

Endpoints

  • Get Hadith by Global Number:

    GET /hadith/:hadithNumber

    Retrieves a single Hadith by its global unique identifier.

    Example: /hadith/1

    Example: /hadith/7563

  • Get Hadith by Book Number and Number within Book:

    GET /book/:bookNumber/hadith/:hadithNumber

    Retrieves a single Hadith by its specific book number and its number within that book.

    Example: /book/1/hadith/1

Important Note for Deployment

For the API to work correctly after deployment, especially on platforms like Vercel, you need to ensure the all_hadiths.json file is generated and available.

Please run the merge script locally before deployment:

cd albukari
node merge_hadiths.js

This will create (or update) the all_hadiths.json file in the albukari/json/ directory, which the API server reads.
